# Expert Attribution

## Author
**Philip Bankier** — Founder, Kairox AI

## Methodology

### Research Sources
This skill is built on established questioning frameworks from multiple domains:

1. **Leadership** — Maxwell's *Good Leaders Ask Great Questions* (8 principles, listening levels)
2. **Critical Thinking** — R.W. Paul's Six Types of Socratic Questions
3. **Innovation** — Berger's *A More Beautiful Question* (Why → What If → How)
4. **Sales Research** — Rackham's SPIN Selling (35,000 call study)
5. **Coaching** — Whitmore's GROW Model (Goal → Reality → Options → Will)
6. **Problem-Solving** — Toyota's 5 Whys
7. **Creative Process** — Gregersen's Question Burst (MIT Sloan)
8. **Education** — Bloom's Taxonomy (6 depth levels)

### Design Principle
The skill does NOT teach the model these frameworks — a SOTA model already knows them from training data. Instead, it provides the **metacognitive layer** that humans have naturally: when to ask, what depth, when to stop, and what's missing.

### Maintenance
- Framework selection should be reviewed as new AI agent interaction patterns emerge
- Anti-patterns list should grow based on observed agent failures
- Pacing rules may need adjustment as models become more capable at reading conversational signals
